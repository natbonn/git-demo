# git-demo
Git Demo Project
Αυτό το project δημιουργήθηκε για να εξασκηθώ στη χρήση του Git και του GitHub μέσα από το IntelliJ.
Περιλαμβάνει ένα απλό Java πρόγραμμα που τυπώνει μήνυμα στην οθόνη και χρησιμοποιείται ως demo για:

δημιουργία Git repository

staging & commit αλλαγών

σύνδεση με remote repository

push στο GitHub
